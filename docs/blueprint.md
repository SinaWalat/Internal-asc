# **App Name**: Message Panel

## Core Features:

- Message Input: A text input field for users to type their messages.
- Submit Button: A button to submit the entered message.
- Confirmation Alert: Display a confirmation alert upon successful message submission.
- Error Handling: Display appropriate error messages if the message submission fails.

## Style Guidelines:

- Primary color: Dark slate gray (#334155) for a modern and subdued feel.
- Background color: Very light gray (#F9FAFB) for a clean backdrop.
- Accent color: Soft indigo (#818CF8) for interactive elements, to indicate activity.
- Body and headline font: 'Inter' for a clean and modern sans-serif aesthetic.
- Use Shadcn components to structure the panel with clear spacing and alignment.
- Subtle transition animations for the confirmation alert.
